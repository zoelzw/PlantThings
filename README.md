# PlantThings
## Nexus 
1. Set up Arduino: download `Adafruit SCD30` and all dependencies 
2. Sensor Wiring: 
    - SCD30: 
        - VIN -- pin 3V (bottom right 2)
        - GND -- GND J4 (bottom right 3)
        - SDA -- pin 17
        - SCL -- pin 16
    - LEDs: 
        - 12V -- PIN 19,20
        - GND -- J4
